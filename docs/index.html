üîÑ Recent API Updates

All examples have been updated to use the current API syntax:

  • addRangeRule() now uses object syntax: { min: 1, max: 10, type: 'integer' }
  • Reference system uses %@variable% syntax directly
  • Enhanced documentation with comprehensive JSDoc comments
  • GitHub CDN integration for web deployment
// CDN Integration - Ready for Production <script src="https://cdn.jsdelivr.net/gh/videlais/story-grammar@main/dist/story-grammar.bundle.js"></script> // Modern API Usage const parser = new StoryGrammar.Parser(); // Range rules with object syntax parser.addRangeRule('level', { min: 1, max: 20, type: 'integer' }); // Weighted probability distributions parser.addWeightedRule('rarity', ['common', 'rare', 'legendary'], [0.7, 0.25, 0.05]); // Template rules with automatic reference handling parser.addTemplateRule('character', '{name} the Level {level} {class}', ['name', 'level', 'class']); // Reference previous values with %@variable% syntax const story = parser.parse('%@character% discovers %rarity% treasure!');
