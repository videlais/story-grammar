This example demonstrates dynamic rule generation using function-based rules in Story Grammar. Unlike static rules that contain fixed arrays, function rules generate values dynamically each time they're called.

Static vs Function Rules

Static Rule: weather: ['sunny', 'rainy', 'cloudy']

Function Rule: randomNumber: () => [Math.floor(Math.random() * 100).toString()]

Live Examples

Implementation Code

// Create parser and add static rules const parser = new Parser(); parser.addRule('character', ['Alice', 'Bob', 'Charlie']); parser.addRule('location', ['forest', 'castle', 'tavern']); // Add function rules for dynamic content parser.addFunctionRule('randomNumber', () => { return [Math.floor(Math.random() * 100).toString()]; }); parser.addFunctionRule('diceRoll', () => { const roll = Math.floor(Math.random() * 20) + 1; return [`${roll} (d20)`]; }); parser.addFunctionRule('timestamp', () => { const now = new Date(); return [now.toLocaleTimeString()]; }); // Use in templates const result = parser.parse('%character% rolls %diceRoll% at %timestamp%');
